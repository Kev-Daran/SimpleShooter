# SimpleShooter
Part of the Unreal course with cpp

## Controls 

### Keyboard and Mouse:
W: Move forward
A: Move Left
S: Move Back
D: Move Right
LMB: Shoot
Space: Jump

### Gamepad:
Left Stick: Movement
Bottom face button: Jump
Right Trigger: Shoot

## Objective:
Clear the hallway from all the enemy soldiers
