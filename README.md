# SimpleShooter
Part of the Unreal course with cpp

## Controls 

### Keyboard and Mouse:
W: Move forward<br>
A: Move Left<br>
S: Move Back<br>
D: Move Right<br>
LMB: Shoot<br>
Space: Jump<br>

### Gamepad:
Left Stick: Movement<br>
Right Stick: Camera Movement
Bottom face button: Jump<br>
Right Trigger: Shoot<br>

## Objective:
Clear the hallway from all the enemy soldiers
